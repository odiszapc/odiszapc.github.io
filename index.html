Most interesting projects

Cloud archived file storage

Distributed file storage (similar to Amazon S3) written on Java, uses Cassandra (http://cassandra.apache.org/) as a storage backend.

When started in 2013 it was the largest Cassandra cluster in the world: 140+ TB, 70+ nodes, over 3 billions of objects

My role: author, architect

My responsibilities:
  • — Design and development of new features (Java)
  • — Team lead and architect
  • — Support and maintenance (Centos on vSphere and physical machines)
  • — Cassandra maintenance, cluster rebalancing
  • — Design, implementation and testing of MapReduce tasks (simple and chainable) for Hadoop on Java with Cassandra as a datasource with Thrift protocol

Cloud solution for video streaming

Distributed video-streaming platform for remote embedded cameras to stream in and for users to view the result transmuxed video over MPEG-DASH and RTMP.

Stack: Java, C++, Consul, Mongo, gRpc, Redis, RabbitMQ, Netty, WebSockets

My role: architect, technical leader

My responsibilities:
  • — Design and development of new features (Java)
  • — Team lead and architect
  • — Integration with streaming server (C++, libuv, Protobuf, Netty)
  • — Planning sprints and preparing new features presentation for management team
